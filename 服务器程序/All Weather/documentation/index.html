Description

“All weather” is a elegant solution to obtaining weather information. The script auto-detects the user’s location via IP with fall-back. The application show the current conditions, next 24 hours (hour by hour) and up to 7 days forecast. Some of the options for forecast is humidity, wind position and speed, cloud cover, rain possibility and more. For 24 hours and 7 day is forecast using nice and clean responsive slider. For better experience weather icons are animated. Own Database with more than 10,000 thousand cities around the world, with their coordinates.  Also, the search engine is very convenient, which is autocomplete. The graph shows the temperature and probability of precipitation for the next 12 hours, but it is possible to add more options. Clean design and very easy to install and edit, without any problems. Work with all major browsers. Your visitors will be happy to know right away how to plan their schedule so the weather will be on their side.

How to install

  1. Extract zip
  2. Create a database on your web server, as well as a MySQL
  3. Find and "config.php then edit the file and add your database information.
  4. Import "location.sql" file from "db" folder to your database
  6. To get API you have to register on https://developer.forecast.io/register 
  7. Upload the script files on your web server
  8. Congratulations, you are now ready!

Options & Customisation

Possible options for:

  • Languages  

Example: Change the default language (English) to Spanish.

- Open and edit "common.php" file on line "64" and raplce "$lang_file = 'lang.en.php';" with "$lang_file = 'lang.es.php';"

Example: Edit the language file.

- Open "lang" folder and edit "lang.en.php" file if you want.

  • Hightcharts - Here you can find the graphics options in which you can add new lines, bars, columns and more. Also you can increase the time that the forecast is shown in the graph (default is 12 hours) but you can change it, and 24 hours.

Example:

// Get weather information for next 12 and added to chart
for($i=0;$i<12;$i++) { // Default is 12 hours but it could be 24
   $chartTemp[] = ceil($forecast->hourly->data[$i]->temperature); // Temperatur
   $unixtime = $forecast->hourly->data[$i]->time;
   $chartTime[] = iconv('Windows-1252', 'UTF-8//TRANSLIT',strftime("%a <br> %H:%M", $unixtime)); // Datetime
   $chartRainPos[] = $forecast->hourly->data[$i]->precipProbability * 100; // Rain possibility in percent
}
  • API Forecast - You can find all the options for forecasts, explained in detail what each option does and how to work with it. The documentation is very detailed and also flexible and you can add more options to the script.
